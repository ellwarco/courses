Formation

Audience

Designer & développeurs souhaitant travailler ensemble avec des applications web. Le but est de favoriser la collaboration rapide & le partage d’informations avec les autres membres de l’équipe.

Format

Formation collective d’une journée, articulée autour de modules. Chaque module comporte un cours théorique et sa mise en pratique.

Si vous avez déjà une idée de produit, n’hésitez pas à prevenir le formateur afin d’améliorer ensemble votre travail.

Pré-requis

  • Papier et crayon
  • Connaissances de bases en HTML, CSS & JavaScript
  • Option : Connaissance de l’utilisation d’un terminal

Objectifs

  • Partager rapidement les informations avec l’équipe
  • Organiser et prioriser le développement
  • Prototyper des solutions évolutives
  • Connaître les outils, services et méthodologies couramment utilisés

Programme

Module 1

Organisation

Pour la réussite d’un projet, il est important de créer des ponts entre designer, développeurs et les autres membres de l’équipe. Quels sont les outils pour travailller ensemble et avoir un language commun ?

Module 2

Développement

Ce module présente différentes notions très utiles pour avoir des projets web évolutifs :

  • conventions de codes
  • performance
  • développement par composants
  • revue de code
  • documentation

Intervenant

David Leuliette

Designer & développeur front-end, David accorde une attention particulière à la qualité du code, dans le but de créer des produits évolutifs & respectueux des standards du web open source.

Il partage ses outils en opensource & ses bonnes pratiques front-end.
